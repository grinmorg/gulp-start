## Особенности сборки
- SASS (SCSS)
- Panini (для создания шаблонов, условий и др. в Html)
- Минификация изображений, стилей и скриптов
- Кеширование изображений во избежании повторного сжатия
- Автопрефиксер css
- Спрайты


## Файлы:
- build - файлы для продакшена
- src - файлы для разработки

## Для запуска сборки:
1) Установить окружение Node.js (LTS)
2) Открыть консоль в корневой папке
3) npm install --global gulp-cli
4) npm i
5) copy .env.example .env
6) получить API ключ от TinyPng (используется для сжатия изображений)
7) gulp
